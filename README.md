# Webspoons Test

## Introduction

A simple Rails app with a list of users and their attributes


## Requirements

- Ruby version 2.7.2

- Rails version 6

- Postgres

## Installation

- Clone the repository and get inside it:

```
git clone git@github.com:JamesOkunlade/webspoons_test.git

cd webspoons_test
```

- Install all gems:

```
bundle install
```

- Create a postgres database:

```
rails db:create
```

- Migrate database

```
rails db:migrate
```

- Seed users

```
rails db:seed
```

- Start the server

```
rails s
```
